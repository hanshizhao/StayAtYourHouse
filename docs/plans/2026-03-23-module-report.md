# 统计报表模块设计

## 模块概述

提供收支统计、房源概览、利润排行、催收统计等报表，帮助房东全面掌握经营状况。

## 报表类型

| 报表 | 说明 |
|------|------|
| 收支统计 | 按月/年统计房租收入、水电收入、总成本、净利润 |
| 房源概览 | 总房源数、已出租数、空置数、空置率 |
| 房间利润排行 | 按利润排序，快速看出哪些房间最赚钱 |
| 催收统计 | 本月应收、已收、未收、逾期金额 |

| 房源成本 | 期间内所有已出租房间的租入成本 × 出租月数 |
| 净利润 | 房租收入 + 水电收入 - 房源成本 |

## 业务规则

1. **收支统计**：
   - 按年显示，可切换月份查看详情
   - 只统计已收款的账单

2. **房源概览**：
   - 实时统计当前状态
   - 空置天数从状态变为"空置"开始计算
   - 点击"入住"跳转到入住办理页面

3. **利润排行**：
   - 月利润 = 出租定价 - 租入成本
   - 年利润 = 月利润 × 12
   - 亏损房源用红色标记提醒

4. **催收统计**：
   - 按月统计，默认显示当前月
   - 应收金额 = 该月所有到期账单的应收总额
   - 已收/待收/宽限/逾期按账单状态分类汇总
