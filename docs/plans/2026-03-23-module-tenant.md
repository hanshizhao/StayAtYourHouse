# 租客管理模块设计

## 模块概述

管理租客信息，支持入住办理、退租结算，记录租住历史。

## 租客信息

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| name | string | 姓名 |
| phone | string | 手机号 |
| id_card | string | 身份证号 |
| gender | enum | 性别：male(男)/female(女) |
| emergency_contact | string | 紧急联系人（可选） |
| remark | string | 备注 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

## 租住记录（合同）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| tenant_id | int | 租客ID |
| room_id | int | 房间ID |
| check_in_date | date | 入住日期 |
| lease_type | enum | 租期类型：monthly(月租)/half_year(半年)/yearly(一年) |
| contract_end_date | date | 合同到期日（自动计算） |
| monthly_rent | decimal | 月租金 |
| deposit | decimal | 押金金额 |
| deposit_status | enum | 押金状态：received(已收)/refunded(已退)/deducted(已扣) |
| deduction_note | string | 扣除说明 |
| status | enum | 状态：active(在租)/terminated(已退租) |
| check_out_date | date | 退租日期 |
| settlement_amount | decimal | 结算金额 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

## 合同到期日计算

| 租期类型 | 计算方式 |
|----------|----------|
| 月租 | 入住日期 + 1个月 - 1天 |
| 半年 | 入住日期 + 6个月 - 1天 |
| 一年 | 入住日期 + 12个月 - 1天 |

## 退租结算

**触发时机**：房东在租客详情页点击"退租"

**结算公式**：
```
结算金额 = 剩余房租 + 押金 - 未结水电费
剩余房租 = 月租金 / 30 × 剩余天数
剩余天数 = 从退租日到下次应收租日的天数
```

**结算金额含义**：
- 结算金额 > 0：退还给租客
- 结算金额 < 0：租客需补交

**押金处理逻辑**：
- 入住时：押金状态为"已收"
- 退租结算时：
  - 正常退租 → 押金全额退还，状态改为"已退"
  - 需要扣除 → 状态改为"已扣"，记录扣除说明

## 业务规则

1. 一个租客可以有多条租住记录（历史）
2. 一个房间同一时间只能有一条"在租"状态的记录
3. 入住办理时：
   - 选择空置房间
   - 填写合同信息
   - 上传合同图片（保存到房间的 contract_image 字段）
   - 房间状态自动改为"已出租"
4. 退租办理时：
   - 自动计算结算金额
   - 处理押金
   - 房间状态自动改为"空置"
   - 房间的合同图片清空
   - 租住记录状态改为"已退租"
