# 水电抄表模块设计

## 模块概述

管理水电表抄表、费用计算、水电账单催收，与房租账单独立管理。

## 水电单价设置

**每个房间可单独设置水电单价**（在房间信息中配置）：

| 字段 | 类型 | 说明 |
|------|------|------|
| water_price | decimal | 水费单价（元/吨） |
| electric_price | decimal | 电费单价（元/度） |

**系统设置中可配置默认值**，新建房间时自动带入，后续可单独修改。

**抄表时使用房间的单价计算费用**。

## 费用计算

```
水用量 = 本次水表读数 - 上次水表读数
电用量 = 本次电表读数 - 上次电表读数
水费 = 水用量 × 房间水费单价
电费 = 电用量 × 房间电费单价
总金额 = 水费 + 电费
```

**注意**：单价使用房间信息中配置的单价，不同房间可以不同。

## 业务规则

1. **抄表流程**：
   - 只对"已出租"状态的房间进行抄表
   - 首次抄表时，上次读数为0或手动输入
   - 抄表时可选择是否同时生成水电账单

2. **账单生成**：
   - 抄表后可选择立即生成水电账单
   - 账单自动关联当前租客

3. **催收流程**：
   - 与房租账单催收逻辑相同
   - 支持"成功/宽限/拒付"三种结果
   - 宽限到期后在首页提醒

4. **退租结算**：
   - 退租时如有未结清水电账单，需先结清
   - 水电费计入退租结算公式
